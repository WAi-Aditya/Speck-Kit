# Skill Matrix Module – Specification

## 1. Module Objective
The Skill Matrix module enables the organization to capture, maintain, and analyze employee skill data within the ITP platform.

The module supports:
- Employee development
- Project allocation
- Workforce capability mapping

---

## 2. Skill Structure

Skills will be organized as:

Skill Category
- Development
- QA
- Cloud
- Communication

Sub Category
- Frontend
- Backend
- Databases
- Automation

Skills
- React
- Java
- SQL
- Selenium
- Azure

Each skill will include:
- Skill name
- Skill description
- Skill category
- Sub category

---

## 3. Skill Proficiency Levels

Beginner
Basic understanding of the skill.

Intermediate
Can work independently with guidance.

Advanced
Strong hands-on experience.

Expert
Recognized subject matter expert.

---

## 4. Skill Rating Mechanism

Skills will be evaluated using:

- Self Assessment
- Manager Assessment
- Peer Validation
- System Generated Score

Final score will be calculated using weighted average.

---

## 5. Skill Assessment Methods

Skill validation can occur through:

- Technical assessments
- Certification upload
- Project experience
- Periodic evaluation

---

## 6. Employee Skill Profile

Employees can:

- Add skills
- Update skill levels
- Upload certifications
- Track skill progress
- View skill dashboard

---

## 7. Manager/Admin Controls

Managers can:

- Approve skills
- Modify skill ratings
- Manage skill taxonomy
- View team capability

---

## 8. Reporting & Insights

Reports available:

- Skill Gap Analysis
- Team Skill Distribution
- Project Skill Matching
- Organization Skill Heatmap

---

## 9. Integration

The module integrates with:

- Employee Profile Module
- Project Allocation System
- Performance Review System
- Learning & Development

---

## 10. Workflow

Employee adds skill → Manager validation → Skill approved → Updated in skill matrix → Visible in reports
